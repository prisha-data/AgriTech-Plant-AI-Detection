## Project Overview 
### Project 1: Potato leaf bight detection 
Developed an image classification model to distinguish between healthy potato leaves and those affected by common fungal pathogens.
* **Classes identified:** Healthy, Early blight(*Alternaria soalni*), and Late blight(*Phtytophthora infestans*).
* **Tools:** Google Teachable Machine, Tensor flow and google colab.

## Project 2: Model Training and Validation work flow
Documents the end-to-end development cycle, including data labeling and accuracy validation.
* **Problem solving:** Successfully optimized the training workflow by leveraging cloud-based resources (**Google Colab**) to work around local **hardware constarints**.

## Skills & Technologies Demonstrated
* **Machine Learning:** Image Classification, Supervised Learning.
* **Technical Tools:** Google Teachable Machine, Google Colab.
* **Domain Knowledge:** Plant Pathology and symptom identification for potato crops.
* **Scientific Documentation:** Preparing technical reports for agricultural AI applications.
